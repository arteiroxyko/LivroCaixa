Image

Image(string file, float x, float y [, float w [, float h [, string type [, mixed link]]]])

Versão

1.1

Descrição

Coloca uma imagem na página. O canto superior-esquerdo deve ser informado. As dimensões podem ser especificadas de diferentes formas:
  • largura e altura (informadas na unidade selecionada pelo usuário)
  • uma dimensão, a outra será calculada automaticamente de forma a manter a proporção original
  • sem dimensões, neste caso a imagem será colocada em 72 dpi
Os formatos suportados são JPEG e PNG.

Para JPEG, todos os tipos são permitidos:
  • gray scales (tons de cinza)
  • true colors (24 bits)
  • CMYK (32 bits)
Para PNG, são permitidos:
  • tons de cinza com pelo menos 8 bits (256 n&iacite;veis)
  • cores indexadas
  • true colors (24 bits)
mas não são suportados:
  • Interlacing
  • Alpha channel
Se uma cor transparente é definida, ela vai ser levada em conta (mas só será interpretada pelo Acrobat 4 e versões superiores).
O formato pode ser especificado ou inferido pela extensão do arquivo.
É possível colocar um link na imagem.

Nota: se a mesma imagem é usada várias vezes, só uma cópia vai ser embutida no arquivo.

Parâmetros

file
Nome do arquivo que contém a imagem.
x
Abscissa do canto superior-esquerdo.
y
Ordenada do canto superio-esquerdo.
w
Largura da imagem na página. Se não informada ou zero, ela será calculada automaticamente.
h
Altura da imagem na página. Se não informada ou zero, ele será calculada automaticamente.
type
Formato da imagem. Os possíveis valores são (diferenciando maiúsculas e minúsculas): JPG, JPEG, PNG. Se não informado, o tipo será inferido pela extensão do arquivo.
link
URL ou identificador retornado pela função AddLink().
